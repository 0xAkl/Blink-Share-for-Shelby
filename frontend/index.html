Core features

Everything you need,
nothing you don't

Six primitives that give you total control over every file you share.

🔐
End-to-End Encryption
AES-256-GCM encryption runs entirely in your browser using Web Crypto API. The decryption key lives only in the URL fragment — never transmitted to any server, ever.
const key = await WebCrypto.generateKey({ name: "AES-GCM", length: 256 })
const enc = await encrypt(file, key) // never leaves browser
const url = `/d/${fileId}#${keyFragment}` // key in fragment only
⏱
On-Chain Expiry
Expiration is written to the contract at upload time. Once the timestamp passes, validateAccess() reverts permanently.

🛡️
Password Protection
Only keccak256(salt, password) is stored on-chain. The plaintext password never touches the blockchain.
👛
Wallet Allowlist
Restrict access to specific wallets. Recipients prove ownership via cryptographic signature — no passwords, no friction.
⬡
Shelby Distributed Hot Storage
Erasure-coded blobs replicated across 3+ Shelby nodes globally. Geomi API key authentication gives you dedicated node access and higher throughput.

One-Time Links

Pay-Per-Upload
